EX2 Release History

3/23/2022
EX2 v1.4-alpha (The Names Update), is released, bringing:

  • Update names! if you look at the updates, it will now have a name, for example, "EX2 V1.2-alpha ("UPDATE NAME HERE")", 1.0 alpha is the first update, 1.1 alpha is the letters & words update, etc.
  • added humans.txt, think of it as a credits file
  • fixed bug where the EX2 release history button had an underscore still

3/21/2022
EX2 V1.3-alpha is released, it brought quite a few new things to it such as:

  • Apple touch icons: if you add this site as a shortcut on an apple home screen, it now has its own icon.
  • android touch icons
  • windows live tile icons
  • Removed the underscore line underneath the text on the buttons at the bottom of page for switching between history and main page

3/19/2022
EX2 v1.2-alpha (The Style Update) is the first major update released, it brought TONS of stuff to it, including:

  • lots of CSS (styles) were added making it look a TON better
  • a release history page that includes descriptions of all the releases
  • a button at the bottom of the main page to change to the release history, and from the release history to the main page

3/18/2022
EX2 v1.1-alpha (The Letters & Words Update) is released, adding:

  • changed font to arial, sans-serif
  • corrected placement of style tags
  • added utf-8 character encoding to prevent character breakage.

3/11/2022,
EX2 v1.0-alpha (The First Update) is the first open source release, bringing with it:

  • Fixed bug where a square around the clock was a little bit greener than the whole background
  • made the code less messy
  • added an embed title and description for things such as texting and etc.
  • checked for any more bugs
